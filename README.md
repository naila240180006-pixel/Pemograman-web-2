# 👟 Stok Sandal - Laravel CRUD Project

Project ini adalah aplikasi **CRUD (Create, Read, Update, Delete)** sederhana untuk mengelola data stok sandal menggunakan framework Laravel.

---

## 🚀 Fitur Aplikasi

- Menampilkan data sandal
- Menambah data sandal baru
- Mengedit data sandal
- Menghapus data sandal
- Validasi input sederhana

---

## 🛠️ Teknologi yang Digunakan

- PHP
- Laravel Framework
- MySQL / MariaDB
- HTML, CSS
- Bootstrap (optional)

---

## 📦 Cara Instalasi

1. Clone repository ini
```bash
git clone https://github.com/username/repo.git
Masuk ke folder project
cd stok_sandal
Install dependency
composer install
Copy file .env
cp .env.example .env
Generate key
php artisan key:generate
Setting database di file .env
Jalankan migration
php artisan migrate
Jalankan server
php artisan serve
📊 Fitur CRUD
➕ Create

Menambahkan data sandal baru (nama, ukuran, stok, dll)

📄 Read

Menampilkan semua data sandal dalam tabel

✏️ Update

Mengedit data sandal yang sudah ada

❌ Delete

Menghapus data sandal dari database
